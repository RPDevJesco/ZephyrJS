🎮 Welcome to GameDev Made Easy!

This demo showcases the XSidebar component in action. The sidebar provides a collapsible navigation experience with smooth animations, responsive behavior, and full accessibility support.

✨ XSidebar Features

📱 Responsive Design

Automatically switches to overlay mode on mobile devices. Try resizing your browser window!

⌨️ Keyboard Accessible

Press Escape to close overlay mode. Full keyboard navigation support built-in.

🎯 Active State Management

Automatically tracks active navigation items and dispatches navigation events.

🎨 Smooth Animations

Configurable animation duration with CSS transitions. No JavaScript animation overhead.

🔄 Push & Overlay Modes

Push mode adjusts content layout. Overlay mode floats over content with backdrop.

⚡ Event-Driven API

Emits sidebar-expand, sidebar-collapse, and sidebar-navigate events for integration.

🎯 Current Navigation State

Click on navigation items to see how the XSidebar component tracks active states and emits navigation events.
